APACHE2

Paso - 1

Actualizamos los paquetes con el comando: sudo apt - get update.

Paso - 2

Instalamos el servidor web Apache con el comando: sudo apt - get install apache2

Paso - 3

Abrimos el fichero: sudo nano /etc/apache2/apache2.conf

Añadimos en la parte inferior: ServerName dominio_del_servidor_o_IP

Paso - 4

Revisamos los errores de sintaxis: sudo apache2ctl configtest.

Paso - 5

Reiniciamos Apache: sudo systemctl restart apache2.
