APACHE2

Paso - 1

Actualizamos los paquetes con el comando:

sudo apt-get update

Paso - 2

Para instalar el servicio Apache2, utilizamos el siguiente comando que vemos a continuación:

sudo apt-get install apache2

Paso - 3

Por defecto el servicio trae el sitio habilitado: /etc/apache2/sites-enabled/000-default.conf
Copiamos el fichero con el comando 'cp' con el nombre que quieras darle para tu sitio web, ej: mysite.com.conf

Después configuramos el archivo principal
Abrimos el fichero: sudo nano /etc/apache2/apache2.conf para configurar los parametro necesario que necesites.

Paso - 4

Revisamos los errores de sintaxis:

sudo apache2ctl configtest

Paso - 5

Reiniciamos Apache: sudo systemctl restart apache2
